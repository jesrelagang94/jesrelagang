## Project Description
Automated billing and reminder system that manages payment schedules, sends automated reminders to tenants, tracks payment status, and generates billing reports. Integrates with accounting software and communication platforms for seamless payment management and collections.

## Technologies Used
N8N, Google Sheets, Twilio, SendGrid, Slack, QuickBooks API, Webhook Integration

## Key Results
- Automated daily reminder notifications for upcoming bills
- 85% reduction in late payments
- Real-time payment tracking and reporting
- Multi-channel notifications (Email, SMS, Slack)
- Automated billing report generation
- Integration with accounting systems for seamless workflow
